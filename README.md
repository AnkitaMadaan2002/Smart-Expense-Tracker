# 💸 Smart Expense Tracker

A machine learning-based expense tracking web app built using Streamlit.

## 🚀 Features
- Add, view, and delete expenses
- Monthly filtering of expenses
- Category-wise visualization (Bar & Pie charts)
- Budget tracking system
- 📈 Future expense prediction using Machine Learning (Linear Regression)

## 🛠️ Tech Stack
- Python
- Pandas
- Streamlit
- Scikit-learn
- Matplotlib

## 📸 Project Overview
This project helps users track daily expenses, analyze spending patterns, and predict future expenses using machine learning.

```bash
pip install -r requirements.txt
